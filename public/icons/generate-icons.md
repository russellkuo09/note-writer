# PWA Icons

Place the following files in this directory:
- `icon-192.png` — 192×192px app icon
- `icon-512.png` — 512×512px app icon

## Quick way to generate them

You can use the SVG flower logo from `src/components/Logo.tsx` and export it at the required sizes.

Or, use a free tool like https://realfavicongenerator.net to convert a PNG logo to all required sizes.

For now, the app will work without these icons — they're only needed for the "Add to Home Screen" PWA feature on iOS/Android.
